# IE410 Robotics Project Part A

## Object Manipulation Using Robotic Arms

This repository contains the implementation, documentation, codes, images, and demonstration videos for IE410 Introduction to Robotics Project Part A.

## Project Tasks

- Task 1: Pre-Programmed Pick and Place
- Task 2: Camera-Assisted Pick and Place
- Task 3: Object Handover Between Two Robot Arms
